Token Beam

Sync design tokens (colors) from any web app to Krita in real-time.

How to open

Settings → Dockers → Token Beam

This opens a small panel you can dock anywhere in Krita's interface.

How to use

  1. In your web app (e.g. Polaris, OKPalette, or the Token Beam demo), start a sync session. You'll get a session token like beam://A1B2C3...
  2. Paste the token into the Token Beam panel in Krita.
  3. Click Connect.
  4. Once paired, colors appear as a grid of swatches in the panel. They update in real-time whenever the web app changes.
  5. Click any swatch to set it as your foreground color.
  6. Click Save as Krita Palette to persist the colors as a .gpl palette file (visible in the Palette docker after restarting Krita).

Requirements

  • The Token Beam sync server must be running (default: ws://localhost:8080).
  • A web app with Token Beam integration to pair with.
